Autonomous Guided Robot Project

What is it about?

The Autonomous Guided Robot project aims to create a robot that can navigate a factory floor autonomously, following a path marked by a black line. The robot can also be controlled remotely via a web interface. This project involved several key steps:

  • UART Communication: Implemented a dual UART messaging system for smooth data transmission.

  • WebSocket Server: Set up a WebSocket server using an ESP-01S WiFi module for remote control.

  • Line Tracking Algorithm: Developed using PixyCam2 to detect and follow lines.

Competencies

UART Communication on STM32


  • Gained a deep understanding of UART communication principles, including baud rate configuration, data framing, and error handling.

  • Developed proficiency in configuring and utilizing the STM32 microcontroller's UART peripherals, including register-level programming and interrupt handling.

  • Acquired skills in efficient buffer management, critical for maintaining data integrity and optimizing memory usage.

  • Embraced an iterative approach to software development, recognizing the importance of continuous improvement and refinement to achieve high-performance and reliable communication systems.

    • ESP-01S Webserver Integration


  • Develop the ability to configure the ESP-01S to run a webserver, manage HTTP requests, and return web pages autonomously.

  • Enhance skills in programming the MCU to handle webserver configurations

  • Demonstrate the capability to control external devices, such as LEDs, through web commands, showcasing the integration of webserver functionality with physical hardware.

    • PixyCam2 Line Tracking

  • Learn to integrate the Pixy2 camera with the mobile platform for visual line detection and navigation

  • Develop skills in creating and programming a line tracking algorithm to interpret visual data from the Pixy2 camera and control the platform's movements accordingly.

  • Incorporate barcode recognition to decode digital markers at intersections, dynamically adjusting the robot's path based on detected markers.

  • This enhances the robot's autonomous navigation capabilities in complex environments.
