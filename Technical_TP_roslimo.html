Autonomous Navigation with ROS

What is it about?

In this project, the aim was to explore and utilize the Robotic Operating System (ROS) to develop a robotic autonomous navigation application. The goal was to enable the AgileX Limo to autonomously navigate from waypoint A to B in a mockup environment.

  • ROS Navigation Stack Setup: Configured and implemented the ROS navigation stack to support autonomous navigation capabilities.

  • Mapping: Created a map of the environment using GMapping, which enabled the robot to understand and navigate the surrounding area.

  • Obstacle Avoidance: Fine-tuned parameters to ensure effective obstacle avoidance, enhancing the robot's ability to navigate safely and efficiently.

  • Waypoint Navigation: Developed and tested the system to ensure the AgileX Limo could successfully travel from waypoint A to B autonomously.

Competencies


ROS Proficiency: Gained in-depth knowledge of the Robotic Operating System (ROS) and its navigation stack, including GMapping, Global Planner, Local Planner, and Recovery Behaviours.

Autonomous Navigation: Demonstrated the ability to set up and utilize the ROS navigation stack to enable autonomous movement of the AgileX Limo from waypoint A to B.

Parameter Tuning: Acquired skills in fine-tuning parameters within the costmap yaml file to optimize obstacle avoidance and ensure smooth navigation.
