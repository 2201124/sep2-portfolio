What is it about?

This project focuses on innovating the Evebot Food Print Pen for the healthcare domain, particularly in hospital food preparation kitchens. The goal is to replace hard copy order chits with a digital solution by developing an automatic print pen conveyor system. This system will print food-safe ink orders directly onto serving trays, reducing the carbon footprint and digitizing existing processes.

  • Domain Research: Conducted secondary research on hospital food preparation to identify problem space and opportunities for digitization.

  • System Design: Designed a comprehensive system that integrates the Evebot Print Pen with a conveyor system to automate printing.

  • Project Management: Coordinated team communication and collaboration, using project management tools to track progress and milestones.
