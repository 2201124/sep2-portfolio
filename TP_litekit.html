Lite Kit

What is it about?

In this project, the aim was to design and create a robotic system capable of tracking and following an object while maintaining a safe distance. This was achieved by combining three key components: the PixyCam, STM32 microcontroller, and Parallax Propeller Board.

  • Hardware Integration: Integrated a vision system (PixyCam 2) with the STM32, which has the color tracking algorithm, and with the Parallax Propeller Board, which focuses on processing the ultrasonic and ToF sensors.

  • Object Detection: Configured the PixyCam to identify a specific color and signature associated with the target object.

  • Movement Control: The STM32 microcontroller gathered information about the detected object's position through signature blocks and then transmitted a simplified movement command to the Parallax Propeller Board.

  • Signal Processing: The Propeller Board translated movement commands into appropriate signals (HIGH & LOW) for the motor drivers, enabling the system to effectively track and follow the object.
